Подробное описание

ChessBoard класс наследник стандартного класса QT - QGraphicsScene. Данный класс используется для отрисовки игрового поля. Он является сценой для QGraphicsWidget.

Аргументы
mapматрица, которая хранит фигуры и их расположение на поле.

Методы

◆ get_figure_from()

int ChessBoard::get_figure_from ( QPair< int, int >  coords) const

get_figure_from константный метод, возвращающий тип фигуры, которая находится в ячейке поля с координатами coords.

Аргументы
coordsкоординаты ячейки поля, содержимое которой нужно узнать.
Возвращает
значение int, которое может принимать целочисленные значения от -5 до 5 включительно.

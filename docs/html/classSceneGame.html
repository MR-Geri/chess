Подробное описание

SceneGame наследник стандартного класса QT - QGraphicsScene. Данный класс используется для отрисовки игрового поля. Он является сценой для QGraphicsWidget.

Аргументы
boardобъект класса GameBoard - игровая доска.

◆ is_end_game()

int SceneGame::is_end_game ( )

is_end_game метод вызывается после каждого хода для проверки конца игры. Если возможных ходов нет, значит игра завершилась. Вызывается после каждого хода.

Возвращает
0 - если игра не завершена 1 - если победили белые -1 - если победили черные
